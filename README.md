# Udaya | Tamil Lyricist & AI Music Portfolio

A modern, high-performance portfolio built with Next.js 14, Tailwind CSS, and Framer Motion.

## 🚀 Getting Started

1. **Install dependencies** (already done):
   ```bash
   npm install
   ```

2. **Run the development server**:
   ```bash
   npm run dev
   ```

3. **Visit the site**:
   Open [http://localhost:3000](http://localhost:3000) in your browser.

## 🛠️ Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Smooth Scroll**: React Scroll

## 🎨 Design Features
- Glassmorphism UI
- Neon Glow Effects
- Background Music Toggle
- Scroll-triggered reveal animations
- Typing effects for hero section
- Full-screen animated background shapes
